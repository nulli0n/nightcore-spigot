# About

**NightCore** is a lightweight library that provides a wide set of utilities to make plugin development faster and more efficient.

This library is required for all [NightExpress](https://www.spigotmc.org/resources/authors/81588/)'s plugins.

## Features

**‚úÖ Features Included:**
- ‚úÖ **Software Compatibility**: Spigot and Paper compatible.
- ‚úÖ **Commands API**: Brigadier-like command API, but less tedious and with custom argument support.
- ‚úÖ **Component API**: Create and use custom text components in messages, GUIs, dialogs, items, etc.
- ‚úÖ **Config API**: Read and write custom config files with ease.
- ‚úÖ **Database API**: Handle SQLite and MySQL databases with SQL query builders and data synchronization across multiple servers.
- ‚úÖ **Dialogs API**: Create and show custom dialogs for players.
- ‚úÖ **Economy Integrations**. Handle custom currencies from various plugins with ease.
- ‚úÖ **Item Integrations**. Handle custom items from various plugins with ease.
- ‚úÖ **Permission Integrations**. Handle player's permission data from any permission plugin.
- ‚úÖ **Language API**: Flexible, highly customizable and powerful language API.
- ‚úÖ **Manager API**: Custom manager classes with built-in support for Listeners, Menus and Tasks.
- ‚úÖ **Menu API**: Inventory GUIs with config support and dynamic content.
- ‚úÖ **Block Tracker**: Track blocks placed by players using the native MC's chunk data container.
- ‚úÖ **Bukkit Wrappers**: A bunch of wrappers for bukkit objects (`ItemStack`, `Sound`, `BukkitTask`, etc) that are pleasure to work with.
- ‚úÖ **Player Utils**: Send messages, dispatch commands, add inventory items to players.
- ‚úÖ **Entity Utils**: Modify entity's attributes and equipment.
- ‚úÖ **Location Utils**: Utilities to work with cuboids, locations, block and chunk positions.
- ‚úÖ **Number Utils**: Parse, round, and format integers, doubles and floats with ease.
- ‚úÖ **Placeholder Utils**: Replace custom placeholders in any string fast.
- ‚úÖ **Profile Utils**: Handle player's profiles properly without spamming the Mojang API.
- ‚úÖ **Random Utils**: A better `Random` implementation with various utility methods.
- ‚úÖ **Time Utils**: Utilities to format and convert time and date.
- ‚úÖ **Item Utils**: Decode and encode `ItemStack` to/from NBT tags.
- ‚úÖ **Enum Utils**: Simple enum parser util.
- ‚úÖ **String Utils**: String format utils.
- ‚úÖ **PDC Utils**: Utilities to work with the `PersistentDataContainer`.
- ‚úÖ **Reflection Utils**: Small reflection helper.

**‚ùå Completely Excluded:**
- ‚ùå Garbage libraries like **XSeries**.
